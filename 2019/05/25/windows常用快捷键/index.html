windows常用快捷键

由于我最近写word比较多，就想着记录下来。

Ctrl + A 全选。

Ctrl + B 字体加粗。

Ctrl + C 复制。

Ctrl + D 删除。（直接按Delete也可以）

Ctrl + E 居中对齐。在Excel 2013版本以上新添加的功能，也叫智能填充，可用于数据的合并、提取、拆分等操作。

Ctrl + F 打开“查找和替换”对话框，并定位到“查找”标签上。

Ctrl + G 打开“查找和替换”对话框，并定位到“定位”标签上。

Ctrl + H 打开“查找和替换”对话框，并定位到“替换”标签上。浏览器中，是打开“历史记录”。

Ctrl + i 让选中的文字变倾斜。

Ctrl + J 两端对齐。

Ctrl + K 打开“插入超链接”的对话框。

Ctrl + L 段落左对齐。

Ctrl + M 左侧段落缩进。

Ctrl + N 新建。

Ctrl + O 打开。

Ctrl + P 打开“打印”对话框。

Ctrl+ Q Excel中快速分析数据

Ctrl + R 段落右对齐。

Ctrl + S 保存。

Ctrl + T 首行缩进。

Ctrl + U 给选中的文本添加下划线。

Ctrl + V 粘贴。

Ctrl + W 关闭某个网页。

Ctrl + X 剪切。

Ctrl + Y 与 Ctrl+z 作用相反，有时候撤销过头了，就可以使用Ctrl+y 恢复上一步操作。

Ctrl + Z 撤销刚才的操作。

Ctrl + Tab 切换页面。

Ctrl + Alt + Tab 快速切换当前打开的窗口。（Alt + Tab、win + Tab也可以）

Ctrl + Shift 切换输入法。

Ctrl + 空格 输入法中英文切换。（Shift也可以）

Ctrl + Esc 显示开始菜单。 （直接按 win 也可以显示开始菜单）

Ctrl + F4 关闭多个窗口中的其中一个窗口。

Ctrl + F5 强制刷新。

Ctrl + 0 网页中恢复默认大小。

Ctrl + 1 单倍行距。

Ctrl + 2 双倍行距。

Ctrl + + 浏览器中放大网页字体。在WPS中就是把选中的文本变为下标。

Ctrl + - 缩小网页中的字体。

Ctrl + = 下标。

Ctrl + Shift + = 上标。

Ctrl + Shift + Esc 打开“任务管理器”。

Ctrl + Shift + > 放大字体。（WPS中）

Ctrl + Shift + < 缩小字体。（WPS中）

Ctrl + [ 缩小选中的文本字体。

Ctrl + [ 放大选中的文本字体。

Ctrl + Home 光标快速回到文件头。

Ctrl + End 光标快速移到文件尾。

Ctrl + 拖动文件 复制文件。

win + D 回到桌面。

win + E 打开“文件资源管理器”。

win + L 电脑直接锁屏。

win + M 所有界面最小化。（win + Shift + M 还原刚才小化的界面。）

win + R 打开“运行”。

win + U 打开设置，相当于桌面右击，选择“显示设置”。

win + Tab 显示切换窗口。

win + Ctrl + F 查找计算机。

Alt + Enter 显示选中项目的属性。（Alt + 双击选中的项目）

Alt + Esc 在当前打开的程序之间切换。

Alt + F4 退出当前程序或关闭当前项目。

Alt + A 打开“表格”菜单。

Alt + E 打开“编辑”菜单。

Alt + F 打开“文件”菜单。

Alt + H 打开“帮助”菜单。

Alt + I 打开“插入”菜单。

Alt + O 打开“格式”菜单。

Alt + T 打开“工具”菜单。

Alt + V 打开“视图”菜单。

Alt + space + C 关闭一个窗口。（WPS中）

Shift + 空格 输入法半角和全角切换。

Shift + Delete 永久性删除。

PrtScr 截屏

连续选中可以先选中第一个目标，按住Shift 不放，再点击另一个目标，那它们之间的所有项目都会被选中。

间接选中可以按住Ctrl 不放，再一一点击你想选择的项目。

查看上一级文件夹 Backspace

取消当前任务 Esc

在选中的项目中向前移动 Tab
